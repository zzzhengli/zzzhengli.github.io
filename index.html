Hazardous substance routing Proof of concept

Singapore's NEA publishes approved routes for hazardous substance transportation, but no structured geospatial data exists publicly. I sourced road names from the accompanying spreadsheet, parsed OpenStreetMap data, and manually mapped each segment using geojson.io to reconstruct the route network as usable GeoJSON.

For the routing layer, I researched the Vehicle Routing Problem to account for multiple drivers and delivery destinations with working hour constraints. I experimented with several approaches — running into complexity issues that caused runtimes exceeding 5 minutes — which pushed me to rethink the problem structure and tradeoffs between accuracy and performance.
